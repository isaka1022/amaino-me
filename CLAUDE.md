# amaino.me プロジェクト

井上 周（Amane Inoue）のパーソナルサイト。Astro + TailwindCSS + DaisyUI。

## クイックリファレンス

- **デプロイ**: `site/` で `git push origin master` → Vercel が自動デプロイ（詳細は [AGENTS.md](./AGENTS.md)）
- **本番 URL**: https://amaino.me
- **開発**: `cd site && npm run dev`

## 技術スタック

- Astro 5, TailwindCSS 4, DaisyUI
- Vercel ホスティング
- GA4 計測、Airtable フォーム埋め込み
